# 🛒 Reward Point Shopping System (THB)

ระบบสะสมแต้มจากการซื้อสินค้า (สกุลเงิน: บาท - THB)  
เมื่อผู้ใช้ทำการซื้อสินค้า ระบบจะคำนวณแต้มสะสม และสามารถนำแต้มไปแลก Gift Voucher ได้

---

## 🎯 ความสามารถของระบบ

✅ คำนวณแต้มจากยอดซื้อ (THB)  
✅ มีโบนัสแต้มตามช่วงยอดซื้อ  
✅ เก็บข้อมูลผู้ใช้และแต้มสะสม  
✅ แลกของรางวัลเป็น Gift Voucher  
✅ บันทึกข้อมูลลงไฟล์ JSON อัตโนมัติ  

---

## 🧠 กติกาการให้คะแนน

- ทุก ๆ 25 บาท = 1 แต้ม
- ซื้อครบ 5,000 บาท → โบนัส 50 แต้ม
- ซื้อครบ 10,000 บาท → โบนัส 150 แต้ม

---

| Points | Reward |

│

---

## ▶ วิธีใช้งาน

### 1️⃣ รันระบบ

```bash
python shopping_system.py
```

### 2️⃣ ใส่ Username

หากเป็นผู้ใช้ใหม่ ระบบจะสร้างบัญชีให้อัตโนมัติ

### 3️⃣ เลือกเมนู

1. Purchase  
2. Check Points  
3. Redeem Gift Voucher  
4. View Reward Catalog  
5. Exit
